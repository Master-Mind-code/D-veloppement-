# Run belife-apis in your local environment

In other to run belife-apis via `npm run start:local` you need to define some varibales inside `env.local.json`. Create the file if needed

## Variable to define

```json
{
  "NODE": {},
  "DB_DRIVER": "",
  "DB_NAME": "",
  "DB_USER": "",
  "DB_HOST": "",
  "DB_PASSWORD": "",
  "DB_PORT": ""
}
```
